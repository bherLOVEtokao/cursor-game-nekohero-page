# 猫咪勇者 / Cat Hero

西幻题材 Roguelike 网页小游戏，基于 Vue 3 + TypeScript + Vite。

## GitHub Pages 部署

1. 打开仓库 **Settings** → **Pages**
2. **Build and deployment** → **Source**：选择 **Deploy from a branch**
3. **Branch**：选择 `main`（或默认分支）
4. **Folder**：选择 **/docs**
5. 保存后访问：`https://<用户名>.github.io/cursor-nekohero-page/`
