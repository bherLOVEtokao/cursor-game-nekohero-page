# 猫咪勇者 / Cat Hero

西幻题材 Roguelike 网页小游戏，在猫化的奇幻世界里探索地图、收集装备、挑战魔王。

---

## 游戏类型

- **题材**：西方奇幻 + 猫咪主题
- **玩法**：Roguelike、回合制战斗、地图探索、装备养成
- **平台**：浏览器网页游戏，支持本地存档

---

## 核心玩法

- **支持手机竖屏**：浏览器在手机竖屏下可正常游玩，无需横屏。
- **v1.0.0-beta10 更新**：代码质量优化（shuffle、法术乘数、dayBoost 常量）；测试补充（174 用例、Vitest coverage、damage/round/spawn spec）；横屏基础适配、无障碍（aria-label、role）；文档完善（spells README、beta10 实现计划）。

### 地图与探索

- 从**新手村**出发，穿越层层地点，最终抵达**魔王城**
- 地图采用层级 + 分支结构，每次游戏路径略有不同
- 在地点可进行：**刷怪**、**前往**下一地点、**补给**（回满血魔）、**商店**、**背包**、**魔法设置**、**学习魔法**、**墓地**（新手村）、**挑战魔王**
- 部分地点（如幽暗森林、废弃矿洞）被魔物占据，无法免费补给，需前往城镇

### 战斗系统

- **回合制战斗**：战斗为**自动进行**，无需逐回合选择
- **魔法施放**：在有休息点的地点学习法术，最多启用 5 个；战斗中按优先级**自动施放伤害法术**，消耗 MP（治疗法术已移除）；若某法术被怪物抗性抵抗，该战斗中不再自动选用；遭遇怪物时可在选择战斗/逃跑前打开背包使用道具
- **法术效果**：吸血、debuff（降低怪物攻防速）、抗性降低、战斗开场 buff；纯 buff 法术（25 种）战斗开场自动施加；六元素共鸣：启用 ≥3 同元素法术时该元素伤害 +10%
- **怪物施法**：法师型怪、魔王、魔王干部具备施法能力，消耗 MP
- **遇敌预判**：系统会提示「打不过」或「逃不掉」，便于规划
- **逃跑**：部分情况可尝试逃离战斗
- **魔法与抗性**：武器含火/冰/毒/雷等属性，怪物具有水/火/冰/雷/毒/风/金/土抗性；法术有光/暗属性与元素区分

### 装备与背包

- **防具**：头盔、盔甲、裤子、鞋子、盾牌（共 5 槽）
- **武器**：双槽位，支持剑、弓、枪、杖及多种魔法属性
- **强化**：装备可消耗金币强化（0～10 级），每次提升属性
- **稀有度**：装备有灰/蓝/紫/金颜色区分（普通/稀有/史诗/传说）
- **套装效果**：同系装备 2/4/6 件触发额外属性加成
- 商店购买装备后进入背包存储区，可在背包中**装备**、**卸下**或**强化**
- 装备有随机属性，同一模板的装备数值可能不同
- 背包可扩容，商店可**出售**道具和未装备的武器/防具（收购价 50%）
- 越深入地图，城镇商店出售的装备越强（商店分级）

### 道具

- **恢复类**：恢复 HP、MP（固定值或百分比）
- **属性提升**：永久增加攻击、防御、速度、血量、魔法
- **替死道具**：死亡时自动触发，以半血半魔传送至最近城镇，免于本局结束
- 遭遇怪物时可在选择战斗/逃跑前打开背包使用道具；**战斗开始后**不可使用

### 难度模式

- **轻松模式**：怪物有等级限制，魔王仅出现在魔王城
- **硬核模式**：魔王可能在地图各处随机出现，越接近魔王城、天数越久，出现概率越高

### 多代勇者

- 勇者死亡后，可创建新勇者从上一城镇（或新手村）继续冒险
- **墓地**：新手村可进入，查看历代勇者姓名、等级、属性、击杀数、死因
- **金币继承**：新勇者继承先代部分金币
- **先代加护**：每位先代会将自身属性的一部分加护给当前勇者
- **死亡 3 次**后：商店享折扣、新勇者获得属性加成

### 魔王与图鉴

- 魔王采用**七宗罪**主题（傲慢、嫉妒、暴怒、懒惰、贪婪、暴食、色欲），每局随机其一
- **怪物图鉴**：遭遇过的怪物显示 HP、攻防速、命中、暴击、暴伤及奖励
- 魔王需遭遇后才在图鉴中显示

### 升级与加点

- 战斗获胜获得经验，升级后获得点数
- 每升一级**固定**增加 1 点 maxHP、1 点 maxMP，再弹出加点界面
- 可**手动分配**点数至 HP、MP、攻击、防御、速度、命中、暴击、暴击伤害

### 成就系统

- 独立成就页面，展示成就列表及完成状态
- 示例：首次通关、累计击杀、死亡次数、勇者等级等

### 存档

- 游戏进度自动保存至浏览器本地
- 再次打开页面时可选择**继续游戏**
